# Keramoutsi Guide

Πλήρης responsive στατική ιστοσελίδα / PWA για το Κεραμούτσι Μαλεβιζίου.

## Περιλαμβάνει

- Responsive σχεδιασμό για κινητό, tablet και desktop
- Ελληνικά και αγγλικά με εναλλαγή γλώσσας
- Διαδραστικά φίλτρα σημείων ενδιαφέροντος
- Ιστορικό χρονολόγιο
- Ενότητα ιστορικών προσώπων
- Gallery με lightbox
- Διαδραστικό χάρτη Leaflet / OpenStreetMap
- Φόρμα που δημιουργεί έτοιμο email προς το Visit Malevizi
- Web app manifest και service worker για βασική offline λειτουργία
- Προσβασιμότητα: landmarks, keyboard navigation, skip link, reduced motion

## Εκτέλεση

Η σελίδα ανοίγει απευθείας με διπλό κλικ στο `index.html`. Για πλήρη λειτουργία PWA/service worker, χρησιμοποίησε τοπικό web server:

```bash
python -m http.server 8080
```

και άνοιξε:

```text
http://localhost:8080
```

Ο χάρτης, τα Google Fonts και τα εικονίδια χρειάζονται σύνδεση στο διαδίκτυο στην πρώτη φόρτωση.

## Δημοσίευση

Το περιεχόμενο του φακέλου μπορεί να ανέβει αυτούσιο σε GitHub Pages, Netlify, Cloudflare Pages ή οποιοδήποτε συμβατικό web hosting.

## Σημείωση

Πρόκειται για λειτουργικό πρωτότυπο. Πριν χρησιμοποιηθεί ως επίσημη ιστοσελίδα, χρειάζεται έγκριση του τελικού κειμένου, επιβεβαίωση πρόσβασης/ωραρίων και άδεια χρήσης τυχόν πρόσθετου φωτογραφικού υλικού.
