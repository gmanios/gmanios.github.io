What it does

Four things, done well

01
𐀀

Signs beside syllables

Every sign is paired with its Latin value — or, for ideograms, its meaning (𐂖 → wine). Sign number and Unicode point on hover.

02
𐀱

Compose on a tablet

Click to engrave signs into the clay panel. Each token stacks the glyph over its reading, and a hyphenated transliteration builds live.

03
𐀏

Search & two scripts

Switch between Linear B (syllabograms · special signs · ideograms) and Linear A. Filter or search by value, number or meaning.

04
𐀵

Print & copy

A dedicated print layout outputs the clean tablet and its transliteration. One click copies both glyphs and reading.

Two systems, treated honestly

Linear B & Linear A

Linear B · teal
𐀷𐀙𐀏

Deciphered Greek

Read since 1952. Ventris and Chadwick showed it records an early form of Greek (Mycenaean). The syllabic values here are the established, standard ones.

wa-na-ka — wanax, “king”.

Linear A · ochre
𐘀𐘁𐘂

Still undeciphered

Language unknown. Linear A recorded the Minoan language, which we cannot yet read. The Latin values shown are conventional transcriptions from the shared AB signs.

They let you transcribe signs consistently — but they are not a translation.

Sources. Sign inventories and phonetic values come from the Unicode Standard — the Linear B Syllabary, Linear B Ideograms and Linear A blocks. Glyphs render through the Noto Sans Linear A / B web fonts, so no special system fonts are required. Pure HTML, CSS and vanilla JavaScript in a single file — no frameworks, no build, no tracking.
